# Legacy PHP Version

This folder contains files from the original PHP-based implementation of Glitchless.

## Files
- `analysis.css` - Analysis page styles (deprecated)
- `dashboard.css` - Dashboard styles (deprecated)
- `dashboard.js` - Dashboard JavaScript (deprecated)
- `script.js` - Global JavaScript (deprecated)
- `style.css` - Global styles (deprecated)

## Migration to Next.js

The current active version is in `glitchless-nextjs/` which uses:
- **React 18** for component management
- **Next.js 14** for routing and SSR
- **TypeScript** for type safety
- **Tailwind CSS** for styling (no separate CSS files)
- **Context API** for state management

These PHP files are kept for reference only and are no longer used in the application.
